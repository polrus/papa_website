# Сайт автора — Имя Автора

Стильный одностраничный сайт детского писателя (русский язык).
Стиль: «whimsical & colorful» (тёплый персик + небо).

## Как открыть

Просто откройте `index.html` в браузере. Сборка не нужна — это обычный HTML/CSS/JS.

Для локального просмотра с сервером (необязательно):

```bash
python3 -m http.server 8000
# затем откройте http://localhost:8000
```

## Что нужно заполнить (ищите пометки ✏️ в `index.html`)

| Что | Где |
|-----|-----|
| **Имя автора** | по всему `index.html` — замените «Имя Автора» (Cmd/Ctrl+F) |
| **Слоган** в шапке | секция `hero`, `hero__tagline` |
| **Биография** | секция `about`, блок `about__text` |
| **Факты об авторе** | список `facts` |
| **Описания произведений** | секция `books`, текст `card__desc` |
| **Ссылки на конкретные рассказы** | атрибут `href` у `card__cover` и `card__link` |
| **Цитата / отзыв** | секция `quote` |
| **Контакты** (почта, Telegram, ВК) | секция `contact`, ссылки `chip` |

## Фотографии и обложки

- **Фото автора:** положите `assets/author.jpg`, в `index.html` раскомментируйте `<img>` в блоке `.portrait`.
- **Обложки книг:** см. `assets/covers/README.txt`.

## Содержимое сайта

- Реальные **названия** прозы и стихов взяты со страницы автора на Проза.ру.
- **Тексты, биография, фото и обложки — заглушки.** Пришлите материалы, и их можно вставить.

## Структура

```
index.html        — страница со всеми секциями
css/styles.css    — оформление и тема
js/main.js        — меню, анимации появления, подсветка навигации
assets/           — фото автора и обложки (covers/)
```
